This plugin allows you to add meta information to post/page according to Product and Offer Schemas.
Then you can show it in widget or using shortcode.

== Description ==
More information about Schema - http://help.yandex.ru/webmaster/supported-schemas/goods-prices.xml#supported

== Frequently Asked Questions ==
To use shortcode just paste [xm11_productadvert post_id="POST_NUMBER"].
post_id is optional. If it is not set, metainformation for current page is shown
